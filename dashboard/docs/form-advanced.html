Form Advanced Components

Select 2

Select2 is a replacement for select boxes. It supports searching, remote data sets, and infinite scrolling of results. In order to replace a regular select element with select2 simply provide an attribute called data-toggle="select2". You can specify other options supported by select2 as well. Please refer to documentation here

How to customize the style of select2?

Open file src/scss/custom/plugins/_select2.scss and change the different style to change the background, text or other colors.

Date Range Picker

A JavaScript component for choosing date ranges, dates and times. In order to enable support of date picker on a regular input element, simply provide an attribute called data-toggle="data-picker". You can specify other options supported by datapciker as data attribute as well. Please refer to documentation here

How to customize the style of date-range-picker?

Open file src/scss/custom/plugins/_daterange.scss and change the different style to change the background, text or other colors.

  
                                                
                                                    

                                                        

                                                            

                                                                
                                                                
                                                            

                                                        

            
                                                        

                                                            

                                                                
                                                                
                                                            

                                                        

                                                    

                                                    
            
                                                    

                                                        

                                                            

                                                                
                                                                
                                                            

                                                        

            
                                                        

                                                            

                                                                
                                                                

                                                                     

Switch

Switches can be used as replacement of default checkbox element. The swicthes are pure css based and can replace a regular checkbox into beautiful switch by simply specifying an attribute data-switch="".

How to customize the style of switch?

Open file src/scss/custom/components/_switch.scss and change the different style to change the background, text or other colors.

Input Masks

Masking on form fields and HTML elements is a common need. We have added support for this as well. To simply mask the input element, specify an attribute called data-toggle="input-mask". You can specify the mask format value via attribute data-mask-format="".
